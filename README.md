# Meal Pet Tracker - Mogumogu


## Project Overview
This project is a meal tracking web application with a virtual pet system.

Users can log their meals each day to keep their virtual pet healthy and alive. If meals are skipped, the pet's condition becomes weaker over time. The system is designed to encourage consistent eating habits through simple gamification elements.

---

## Main Features
- Daily meal logging
- Virtual pet status system
- Meal history records
- Pet condition updates based on eating habits
- Streak system for consecutive daily meal logs
